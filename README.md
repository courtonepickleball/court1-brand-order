# court1-brand-mall

코트원 브랜드 상품을 소개하고, 주문서를 통해 견적서를 받을 수 있는 웹 프로젝트입니다.  
사용자는 브랜드 상품을 확인하고, 장바구니에 담아 수량을 조절한 뒤 주문 요청을 보낼 수 있습니다.

## Features
- 상품 목록 표시
- 브랜드 / 카테고리 필터
- 장바구니 추가, 수량 조절, 삭제
- 주문 정보 입력 후 메일 전송

## Installation
```bash
git clone https://github.com/사용자명/court1-mall.git
cd court1-mall
```

## Usage
브라우저에서 `index.html`을 열면 시작 화면이 표시됩니다.  
메인 상품 페이지는 `main.html`, 장바구니 페이지는 `cart.html`에서 확인할 수 있습니다.

## Folder Structure
```text
index.html: 시작 페이지
main.html: 메인 상품 페이지
cart.html: 장바구니 페이지
style.css: 공통 스타일
assets/js/app.js: 동작 로직
assets/js/products.js: 상품 데이터
assets/images: 이미지 파일
```

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js / Python for data processing
